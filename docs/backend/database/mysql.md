## 1. 使用 图形化软件 操作数据库相关

::: tip 说明

使用 图形化软件 workbench 创建表时，要注意的一些字段含义

:::

### 1. dateType 数据类型说明

- ① int 整数
- ② varchar(len) 字符串
- ③ tinyint(1) 布尔值

### 2. 字段的特殊标识说明

- ① PK（Primary Key）主键、唯一标识
- ② NN（Not Null）值不允许为空
- ③ UQ（Unique）值唯一
- ④ AI（Auto Increment）值自动增长
