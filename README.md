## my first blog

我的第一个文档博客，同时连接了 github 和 gitee

### 1. 推送代码到 Github

```xml
<!-- origin 是远程仓库地址 https://github.com/tgx1587900660/tgx-blog.git 的别名 -->
git push origin master
or
git push
```

### 2. 推送代码到 Gitee

```xml
<!-- gitee 是远程仓库地址 https://gitee.com/tgx-1587900660/tgx-blog.git 的别名 -->
git push gitee master
```
